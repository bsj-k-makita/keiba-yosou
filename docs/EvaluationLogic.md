## 全体フロー

```
能力5軸 (0〜100)
  │
  ├─ [1] コース別基準重み取得
  │
  ├─ [2] 条件補正（馬場 + バイアス + ペース）
  │
  ├─ [3] 基礎能力コア計算
  │
  ├─ [4] 過去走シグナル（再現性加点 / 大敗ペナルティ）
  │
  ├─ [5] 合成スコア = 基礎60% + 条件40%
  │
  ├─ [6] レース内相対化（z-score → 35〜85）
  │
  ├─ [7] 展開適合ボーナス（脚質 × ペース × バイアス）
  │
  └─ [8] 最終スコア → 消し判定 → 印・買いラベル
```

## [1] 能力5軸

| 軸 | キー | 意味 |
|----|------|------|
| スピード | `speed` | 前半〜中盤のトップスピード |
| スタミナ | `stamina` | 距離を持ちこたえる消耗耐性 |
| 末脚 | `kick` | ゴール前の瞬発力 |
| 持続 | `sustain` | 高速ラップを維持する能力 |
| パワー | `power` | 重馬場・坂・タフな条件での踏ん張り |

全て 0〜100 の数値。評価ロジックは与えられた数値を条件に応じて重み付けする。

## [4] 基礎能力コア計算

```
基礎能力コア = (5軸平均 × 0.75) + (上位2軸の平均 × 0.25)
```

強みの軸を少し上乗せする設計。純粋な平均より突出した能力が評価されやすい。

## [5] 過去走シグナル

`pastRuns`（直近最大5走）から自動推定。手動入力がある場合は手動優先。

### 再現性スコア（reproducibility01）

過去5走の200m通過秒から展開を分類（瞬発戦/持続戦/消耗戦/高速巡航/バランス）し、最多展開パターンの出現回数で算出。

| 条件 | reproducibility01 | 加点換算 |
|------|------------------|---------|
| 最多パターン 3回以上 | 0.72 | +0.88 |
| 最多パターン 2回 | 0.52 | +0.08 |
| 全パターン散発・走数3以上 | 0.36 | -0.56 |
| データ少ない | 0.45 | -0.20 |
| データなし | — | 0（中立） |

```
再現性加点 = (reproducibility01 - 0.5) × 4
```

### 大敗ペナルティ（riskPenaltyPoints）

「大敗」= 着差 1.5 秒超。「好走」= 3着以内 または 着差 0.5 秒以内。

| パターン | ペナルティ |
|---------|---------|
| 直近3走で大敗 2回以上 ＋ 好走実績あり | -2 点 |
| 直近3走で大敗 2回以上 ＋ 好走実績なし | -4 点 |
| 直近3走で大敗 1回 ＋ 好走なし | -1 点 |
| 直近5走で2桁着順 3回以上 ＋ 好走あり | -1 点 |
| 直近5走で2桁着順 3回以上 ＋ 好走なし | -2 点 |

**好走実績があれば「状況依存の大敗（展開・馬場不向きなど）」と判断し減点を軽減する。**

## [6] 合成スコアと相対化

### 合成スコア（raceAdjustedInput）

```
内在能力スコア = 基礎能力コア + 再現性加点 - 大敗ペナ
合成スコア    = 内在能力スコア × 60% + 条件補正スコア × 40%
```

### レース内相対化

同一レース出走馬の合成スコアを z-score で正規化。

```
相対スコア = 50 + z × 8    ← 35〜85 に収める
```

- 分散が小さい（馬同士の差が僅差）なら min-max にフォールバック
- 同じスコアなら全員 50

## [7] 展開適合ボーナス

脚質 × ペース（3段階）× バイアス の両方を評価し、**低い方の評点を採用**（ボトルネック方式）。

### 脚質 × ペース

| ペース\脚質 | 逃げ | 先行 | 好位 | 差し | 追込 | 自在 |
|------------|------|------|------|------|------|------|
| スロー | ◎ | ◎ | ○ | △ | × | ○ |
| ミドル | ○ | ○ | ◎ | ○ | △ | ○ |
| ハイ | △ | △ | ○ | ○ | ◎ | ○ |

※ スロー = slow / no_front_runner、ハイ = high / many_front_runners

### 脚質 × バイアス

| バイアス\脚質 | 逃げ | 先行 | 好位 | 差し | 追込 | 自在 |
|-------------|------|------|------|------|------|------|
| フラット | ○ | ○ | ○ | ○ | ○ | ○ |
| 前残り | ◎ | ◎ | ○ | △ | × | ○ |
| 差し有利 | △ | △ | ○ | ◎ | ◎ | ○ |
| 内有利 | ◎ | ○ | ○ | △ | × | ○ |
| 外有利 | △ | △ | ○ | ○ | ○ | ○ |

### ボーナス換算

| 評価 | 加点 |
|------|------|
| ◎（PERFECT） | +5 |
| ○（FIT） | +2 |
| △（MAYBE） | 0 |
| ×（BAD） | -5 |

## [8] 最終スコア

```
最終スコア = 相対スコア + 展開ボーナス  （0〜100 にクランプ）
```

---

## 消し判定

以下4条件のうち **2つ以上**該当で「消し」候補。

| 条件 | 内容 |
|------|------|
| 順位低迷 | 最終順位が出走頭数 − 1 位以下 |
| 条件不適合 | 能力 × 今回重みの適合度スコアが 低（0.48未満） |
| 展開不利 | 展開適合が × （BAD） |
| スコア低下 | 条件補正後スコア − 標準スコア ≦ −1.0 |

---

## 買いラベル

| 最終順位 | 条件 | ラベル |
|---------|------|--------|
| 1位 | — | **本命候補** |
| 2位 | — | **対抗候補** |
| 3位 | — | **単穴候補** |
| 4〜5位 | 条件スコア差 ≧ 3 | **穴候補** |
| 4〜5位 | 条件スコア差 < 3 | **相手候補** |
| 6位以下 | — | 消し |
| 消し判定あり | — | 消し（順位より優先） |
| 本命候補 ＋ 2桁着順3回以上 ＋ 好走なし | — | → 相手候補に格下げ |

## 印の付け方

| 最終順位 | 印 |
|---------|-----|
| 1位 | ◎ |
| 2位 | ○ |
| 3位 | ▲ |
| 4〜5位 | △ |
| 6位以下で上位との差が小さい | ☆ |
| それ以外 | （なし） |

### 穴馬ガード

単勝オッズ 50倍超 ＋ 直近3走で大敗1回以上 ＋ 基礎スコア5位以外 → ◎/○ を ▲ に格下げ
